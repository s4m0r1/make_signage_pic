## 使用しているフォント
同梱していませんが[やさしさフォント](http://www.fontna.com/blog/736/)を使用しています。

各自ダウンロードし設置してください。

## コンフィグ一覧

|コンフィグ名|説明|
|:-------|:--|
|FONT|使用フォントの設定|
|COLLOR|横帯のカラー設定|
|LAN_NAME|系統名の設定|
|IP_ADDR|系統のIPアドレス設定|
|HOST_NAME|異常の出ているホスト名|
|LEVEL|WARNで☓,INFOで△,OKで○|
|LEVEL_PING|疎通できるかできないか|
|LEVEL_PING_ENG|上の英語|
|LEVEL_INFO|原因|
